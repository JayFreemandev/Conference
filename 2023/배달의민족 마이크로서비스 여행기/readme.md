**이벤트 전파까지는 오케이 데이터 동기화는?**

- 최종적 일관성
- 데이터는 언젠가는 다 맞추어 진다
- 데이터 싱크 1초~3초
- 문제 발생시 해당 시스템이 이벤트만 재발행
- 대부분 제로 플레이로드 방식 사용
    - 이벤트에 식별자(가게ID)와 최소한의 정보만 발행
    - 이벤트를 받은 시점에 조회 API로 필요한 데이터를 조회 저장
